Choose an App or Create a New App With Heroku

You can choose to work within a previously created version of Jupyter Notebook or create a new instance.

Note: Following setup instructions will be using Heroku, See below for information on configuring a personal server

  • To open an existing instance, select from the list
  • To create a new instance:
    1. Select Create New button
    2. Create a password and enter any required Python packages
    3. Once you've created your app, check the box if you want to use this instance each time you open Jupyter Notebooks

What Can I do?

Using Jupyter Notebooks, you can write and share your code so others can see your work, but that's not all. Because Jupyter is a visual interface for coding languages, you can execute standard code from within the Jupyter environment.

And More!

With Jupyter, you'll be able to:

  • Share documents containing live code
  • Display equations, visualizations, text, and more within your document
  • Clean and manipulate data
  • Use statistical modeling
  • Open .ipynb files directly from the Open in App Menu in data.world
  • View terminals, text editors, data file viewers, and other custom components side-by-side with notebooks in a convenient tabbed view
